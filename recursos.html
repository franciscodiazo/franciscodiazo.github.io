Agrupación de herramientas (OVAs y OVIs) para impartir clases interactivas de Matemáticas, Geometría y Lógica de Programación aplicando aula invertida.

Matemáticas y Geometría

GeoGebra

Plataforma para geometría dinámica, álgebra interactiva, y cálculo. Indispensable para llevar lo abstracto a lo visual.

Ir a GeoGebra →

PhET Simulations

Simulaciones interactivas de ciencias y matemáticas elaboradas por la Universidad de Colorado.

Ir a PhET →

H5P

Sistema para crear y compartir contenido HTML5 interactivo y rico en multimedia, perfecto para evaluaciones formativas.

Conocer H5P →

Pensamiento Computacional

Scratch

Plataforma del MIT que enseña a codificar mediante bloques. Ideal para introducir lógica computacional.

Crear en Scratch →

Pictoblox

Entorno de programación gráfica que agrega Inteligencia Artificial y Machine Learning para estudiantes.

Explorar Pictoblox →

Simuladores de Circuitos

Laboratorios virtuales como Tinkercad para introducir Arduino, electrónica básica y Robótica sin riesgos ni costos de hardware.

Diseñar Circuitos →

Creación de OVAs y OVIs

Canva para Educación

Diseño gráfico simplificado para crear infografías, presentaciones y recursos visuales impactantes (OVIs) de forma rápida.

Diseñar en Canva →

Genially

Plataforma líder para la creación de contenidos interactivos (OVAs), presentaciones animadas, infografías y experiencias Escape Room.

Crear en Genially →

Plataformas de Gamificación

Wordwall

Crea actividades personalizadas e interactivas como cuestionarios, emparejamientos, y juegos de palabras. Excelente para gamificar evaluaciones.

Jugar con Wordwall →

Educaplay

Crea y comparte actividades educativas multimedia interactivas. Cuenta con un amplio repositorio de recursos creados por otros docentes.

Explorar Educaplay →

Kahoot! / Quizizz

Plataformas de aprendizaje basadas en juegos, ideales para la evaluación formativa y el repaso competitivo en el aula invertida.

Gamificar con Kahoot →
