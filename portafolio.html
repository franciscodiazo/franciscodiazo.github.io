Web App Educativa

Sisvot: Democracia Estudiantil

Un sistema robusto diseñado para gestionar procesos electorales dentro de las instituciones educativas. Elimina el conteo manual, previene el fraude y gamifica el proceso democrático para que los estudiantes se interesen en la política escolar.

Arquitectura y Stack:

  • Bases de datos relacionales aseguradas contra inyecciones SQL.
  • Autenticación de usuarios por roles (Admin, Jurado, Votante).
  • Dashboards de resultados en tiempo real y UI/UX adaptable.
